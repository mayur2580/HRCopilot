# HR CoPilot

## Backend setup

From your project root:

Run backend:

```bash
uvicorn backend.main:app --reload --port 8000
```

## Frontend setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```text
http://localhost:5173
```

Backend will run on:

```text
http://127.0.0.1:8000
```
